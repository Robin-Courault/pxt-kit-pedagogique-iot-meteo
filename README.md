Redigez en [Markdown](https://www.markdownguide.org/basic-syntax/) vos rapports dans [./reports](./reports). PDF, Word ... sont proscrits

Redigez en [Markdown](https://www.markdownguide.org/basic-syntax/) vos présentations dans [./presentations](./presentations). PDF, Word, PPT ... sont proscrits
